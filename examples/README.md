# Example Python Files

This directory contains sample Python files for testing CodeMesh functionality.

## Files

- **sample_code.py**: Basic Python file with functions and classes
- **sample_utils.py**: Utility functions demonstrating various patterns
- **complex_sample.py**: Complex code with nested structures and dependencies

## Usage

Upload these files through the CodeMesh web interface to see how the tool analyzes and visualizes Python code structure.

You can also upload them as a ZIP archive for batch analysis.
